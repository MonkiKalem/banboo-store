## ğŸ”§ Installation

### Prerequisites
- Flutter SDK 3.24.2
- Dart SDK
- Node.js v21.6.2
- MySQL

### Clone Repository
```bash
git clone [https://github.com/zakysatya/banboo-store.git]
cd banboo-store
```

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd mobile
flutter pub get
flutter run
```
